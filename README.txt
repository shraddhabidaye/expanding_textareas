Expanding Textareas
======================================================
This module creates integration with Expanding Textareas jQuery Plugin (https://github.com/bgrins/ExpandingTextareas) which is based off of work by Neil Jenkins that can be seen here: http://www.alistapart.com/articles/expanding-text-areas-made-elegant/

For more usage please visit : http://bgrins.github.io/ExpandingTextareas/

Installation
======================================================
1) Download the module into the modules directory (/sites/all/modules/).
3) Install via admin/modules.

Configuration
======================================================
1) Visit Configuration > User Interface > Expanding Textareas Settings
(admin/config/user-interface/expanding-textareas)
to configure module.
2) Add one textarea id per line (not including # sign) or
use * to trigger for all textareas.
